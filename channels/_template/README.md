# {Channel Name}

This is a channel bank template. Copy this directory to create a new channel bank:

```bash
cp -r channels/_template channels/my-channel
```

Then fill in each file with the channel's actual details.

## Files
| File | Purpose |
|---|---|
| `flows.md` | Purchase and cart flows for this channel |
| `channel-contracts.md` | Channel identifiers, headers, unique fields |
| `error-handling.md` | Error format, common errors, retry policy |
