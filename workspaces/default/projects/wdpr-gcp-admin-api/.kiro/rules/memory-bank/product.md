# wdpr-gcp-admin-api - Product Overview

## Purpose
Node.js BFF/API layer for the Gift Card Platform (GCP) administration system.

## Key Features
- Gift card administration APIs
- Balance inquiries and adjustments
- Card lifecycle management

## Target Users
- GCP UI (wdpr-payment-controls-client)
- Gift card administrators
