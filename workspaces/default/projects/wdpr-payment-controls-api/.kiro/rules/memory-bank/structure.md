# wdpr-payment-controls-api - Project Structure

## Directory Organization
- `src/controllers/` — Route handlers and API endpoints
- `src/services/` — Business logic and backend integration
- `src/models/` — TypeScript interfaces and types
- `src/middleware/` — Express middleware
- `src/utils/` — Utility functions

## Architecture
- Part of Config Studio 3-tier architecture
- WebAPI/BFF layer: receives requests from UI, transforms and forwards to backend
