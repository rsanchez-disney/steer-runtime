# wdpr-payment-controls-client - Project Structure

## Directory Organization
- `src/app/` — Angular application root
- `src/app/modules/` — Feature modules (Config Studio, GCP, TRP, PAP)
- `src/app/shared/` — Shared components, services, pipes
- `src/assets/` — Static assets
- `src/environments/` — Environment configurations

## Architecture
- Multi-project Angular application serving Config Studio, GCP, and TRP|PAP
- Part of Config Studio 3-tier architecture (UI layer)
