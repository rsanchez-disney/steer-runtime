# wdpr-payment-controls-client - Product Overview

## Purpose
Angular frontend for the Config Studio payment configuration management platform. Also serves as the shared UI for GCP and TRP|PAP projects.

## Key Features
- Payment configuration management UI
- Gift card platform administration (GCP module)
- Transaction research portal (TRP module)
- Payment admin portal (PAP module)

## Target Users
- Payment operations teams
- Gift card administrators
- Transaction research analysts

## Jira Prefixes
- `DPAY-` (Config Studio, TRP|PAP)
- `GCP-` (Gift Card Platform)
