# Studio Incredibles — Team Context

## Team Overview

Studio Incredibles is the larger development team focused on **100% development** across Advanced Finder, Astro Homepage, Discovery Services, Search UI, HKDL, SEO, and accessibility.

**Jira Board**: https://myjira.disney.com/secure/RapidBoard.jspa?rapidView=4972 (Scrum)
**Jira Prefix**: GIT-
**Sprint Cadence**: 2-week sprints (currently Sprint 388)

## Work Streams

| Stream | Description |
|--------|-------------|
| **Advanced Finder** | UI bugs, accessibility, calendar, map, FAQ uplift, ATMP modal, Lightning Lane, flip cards |
| **Astro Homepage / TXP** | Template work (Explore, Plan, Get Ready, Questions Banner, Plan Banner) |
| **Discovery Services** | Transportation Publisher, Guest Context Service, Capella migration, Kinesis infrastructure |
| **Search UI** | B2B client, Discovery Service integration, environment config |
| **Accessibility (a11y)** | Color contrast, keyboard behavior, focus indicators — high priority |
| **SEO** | Schema markup (video, FAQ), list page meta |
| **HKDL** | Hong Kong Disneyland Finder pages, resource bundles, hero, cross-promo |
| **DCL** | Disney Cruise Line homepage components |
| **OneID JWT v5** | Security migration for DLR and WDW web apps |
| **LiveOps** | Grafana upgrade, Google Play API |
| **Infrastructure** | Redis→Akamai HVC migration, Harness pipeline, Lambda S3 access |
