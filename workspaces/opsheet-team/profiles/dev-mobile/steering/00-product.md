# OpSheet Plus

OpSheet Plus is a mobile application (iOS and Android) for operational management of theme park
entities (attractions, shows, etc.). It provides real-time operational data including:

- Entity status management (open/closed/downtime with reason codes)
- Wait time posting and monitoring across multiple lanes
- Guest count tracking and capacity management
- Performance metrics (dispatch intervals, guests carried, guests per vehicle)
- Occupancy monitoring and capacity percentages
- Schedule and show time management
- System announcements and push notifications
- Tip board and performance summary dashboards

The app is used by operations cast members to manage and monitor park entity data in real time. It
connects to backend VAS (Venue Availability Services) APIs.

Bundle ID: `com.disney.opsheet-plus`

Environments: `latest` (dev), `stage`, `prod`
