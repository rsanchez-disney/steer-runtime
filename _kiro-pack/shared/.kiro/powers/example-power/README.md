# Example Power

Demonstrates the Kiro Powers system.

## Tools

### echo
Echoes back a message with an optional prefix.

**Parameters:**
- `message` (string, required): Message to echo
- `prefix` (string, optional): Prefix to add (default: "Echo:")

**Returns:**
```json
{
  "success": true,
  "output": "Echo: your message"
}
```
