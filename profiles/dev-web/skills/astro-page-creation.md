# Skill: Astro page creation

Use when creating a new Astro page with React components.

## Checklist
1. Create `.astro` file in `src/pages/` (file-based routing)
2. Add frontmatter block for server-side data fetching
3. Import layout and wrap content
4. Import React components with appropriate `client:*` directive
5. Use `client:visible` for below-fold, `client:load` for above-fold
6. Add SEO meta tags via layout props
7. Test page renders correctly with data
