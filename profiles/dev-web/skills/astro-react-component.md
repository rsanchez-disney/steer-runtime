# Skill: React component in Astro

Use when creating a React component for use in Astro pages.

## Checklist
1. Create `.tsx` file in `src/components/`
2. Define TypeScript interface for props
3. Use Vista Web Components for UI elements where possible
4. Implement with functional component + hooks
5. Create `.test.tsx` with `@testing-library/react`
6. Export as named export (not default)
7. Use in Astro page with `client:load` or `client:visible`
