# Skill: UI feature implementation (Angular)

Use this skill when the task is to implement or refactor UI behavior in Config Studio.

## Workflow
1) Identify feature surface (component/template/service/test).
2) Confirm contract assumptions (what fields exist; how to handle missing data).
3) Implement with minimal diff and safe defaults.
4) Update unit tests and any mocks/stubs.
5) Validate UX states: loading/empty/error/success.

## Export-related checklist
- Bubble message matches report type + filters.
- File naming aligns with report type and user selections.
- No UI blocking; async states handled.
