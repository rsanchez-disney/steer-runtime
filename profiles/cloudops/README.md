# CloudOps Profile

Infrastructure strategy, environment planning, and SRE.

## Agents

| Agent | Description |
|-------|-------------|
| infra_planner_agent | Plans environments, sizes infrastructure, assesses feature impact on ops stability |
| config_management_agent | Defines configuration management: version control strategy, secret management, drift detection, DR |
| rca_writer_agent | Produces publishable RCA documents from incident data, logs, and timelines |
