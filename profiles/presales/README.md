# Pre-Sales Profile

Pre-sales discovery, client intake, and project scoping.

## Agents

| Agent | Description |
|-------|-------------|
| presales_agent | Processes client materials, generates discovery questions, project briefs, and feasibility assessments |

## Composed With

This profile works best when combined with:
- `estimation_agent` (from ba profile) — for detailed effort estimation
- `scope_definer_agent` (from ba profile) — for scope definition
- `architecture_agent` (from dev-core) — for technical architecture guidance
- `prd_generator_agent` (from pm profile) — for PRD generation
