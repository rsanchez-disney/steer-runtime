# Design Profile

Design discovery, user research, and UX pipeline.

## Agents

| Agent | Description |
|-------|-------------|
| design_discovery_agent | Orchestrates design discovery: competitive analysis, heuristic audits, vision frameworks |
| user_research_agent | Plans and synthesizes user research: interview guides, personas, journey maps, JTBD |
| usability_testing_agent | Designs usability test scripts, concept validation sessions, surveys |
| design_research_reporter_agent | Produces research plans, desktop research reports, and executive summaries |
| prototype_prompt_agent | Transforms requirements into high-fidelity design prompts for Figma Make, Google Stitch, or v0 |

## Usage

Activate this profile when working on:

- Early-stage product discovery and competitive landscape analysis
- User research planning, execution synthesis, and reporting
- Usability testing protocol design and findings documentation
- Design system prompt engineering for AI prototyping tools
- Stakeholder alignment artifacts (vision frameworks, elevator pitches)
