# Product context — Config Studio

Config Studio provides capabilities to:
- Search/browse clients and configurations
- Compare clients and visualize differences
- Export reports (full / finance / custom) with long-running operations
- Promote / preview configuration changes across environments

## UX principles
- Users need transparent progress and clear outcomes for exports.
- When long-running: show progress, allow cancel if supported, and avoid UI freezes.
- Favor consistent naming for reports and exported files.
