# Skill: Python API endpoint implementation

Use when adding or modifying API endpoints in FastAPI or Flask.

## Checklist
1. Define Pydantic request/response models with validation
2. Create or update router with proper HTTP method and path
3. Implement service layer logic (keep routers thin)
4. Add dependency injection for shared resources
5. Handle errors with appropriate HTTP status codes
6. Add pytest tests covering happy path and error cases
7. Update OpenAPI docs if needed (FastAPI auto-generates)
