# Report Template

The orchestrator generates the final report in this format. Write to `.kiro/inspector-reports/<YYYY-MM-DD>-<HHmmss>-report.md`.

## Template

```markdown

**Target:** <service/path>

**Duration:** <total elapsed time>
**Agents:** <count> specialists completed
```

## Rules

1. Findings are grouped by severity, then sorted by confidence (HIGH first)
2. Deduplicated findings show all contributing agents in "Found by"
3. If an agent was skipped (e.g., no Terraform files → drift_detector skips), mark as "⚠ skipped (no applicable files)"
4. The trend section is populated from yax history; if no prior run exists, omit the section
5. Report filename uses UTC timestamp: `2026-05-02-041530-report.md`
