# wdpr-payment-controls-api - Product Overview

## Purpose
Node.js BFF/API layer for the Config Studio payment configuration management platform. Acts as middleware between the Angular UI and Java backend services.

## Key Features
- REST API endpoints for payment controls
- Request transformation and validation
- BFF (Backend for Frontend) logic
- TypeScript interfaces for type safety

## Target Users
- Config Studio UI (wdpr-payment-controls-client)
- Consumed by Angular frontend
