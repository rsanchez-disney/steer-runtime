# Project Brief: steer-runtime

## Overview
Unified multi-profile agent system for Kiro (AI-assisted development). Provides specialized agents for development, BA/PO, QA, and ops workflows within Disney Payments organization.

## Core Purpose
Package and distribute Kiro agent configurations (prompts, tools, MCP servers) across four profiles: dev, ba, qa, and ops. Agents automate SDLC tasks by integrating with Jira, Confluence, GitHub, Harness, and SonarQube.

## Target Users
- Developers (Config Studio team — Java backend, Node.js API, Angular UI, Flutter/mobile)
- Business Analysts / Product Owners
- QA Engineers / Test Automation
- DevOps / Operations (AI metrics, infrastructure, deployments, code quality)

## Key Repositories (Config Studio)
- `wdpr-config-services` — Java backend services
- `wdpr-payment-controls-api` — Node.js/TypeScript BFF/API
- `wdpr-payment-controls-client` — Angular frontend
