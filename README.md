# steer-runtime

Agent configs, prompts, context, hooks, MCP servers, and workspaces for the steer AI development platform.

## Installation

```bash
koda sync
```

Or via Koda upgrade:

```bash
koda upgrade
```

## Releases

Binary releases are available as `.tar.gz.enc` assets attached to GitHub Releases. They are downloaded and decrypted automatically by `koda sync`.
